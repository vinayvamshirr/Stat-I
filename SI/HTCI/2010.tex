\ques{4}{b}{20} A random observation $X$ belongs to a population having probability desnity function $f(x)$. Describe the likelihood ratio test for testing $H_0 : f(x)=f_1(x)$ against $H_1: f(x) = f_2(x)$ based on a single observation, when
\[ f_1(x) = \frac{1}{\sqrt{2\pi}}e^{-\frac{1}{2}x^2}, \quad -\infty < x < \infty,\]
and
\[ f_2(x) = \frac{1}{2}e^{-|x|}, \quad\enskip -\infty < x < \infty.\]
