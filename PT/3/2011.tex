\ques{2}{a}{20} A random variable $X$ has the probability density function
\begin{equation*}
    f(x) = \begin{cases}
    \frac{1}{2}e^{-|x|}&,\quad -\infty <x<\infty\\
    0 &,\quad otherwise.
    \end{cases}
\end{equation*}
Derive the expression for its characteristic function.
\\\\
\ques{2}{b}{20} State (without proof) the inversion theorem relating to the characteristic function of a continuous random variable.\\
Use this theorem to obtain the probability density function of a random variable having the characteristic function $\Phi(t) = e^{-|t|}$.
